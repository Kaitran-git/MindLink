Test Scenarios

  1. New Homepage Design: Check the welcoming layout with emoji icons and "How it works" section
  2. Skip Challenge Button: Test the demo "Skip Challenge" button in bottom-right corner
  3. Program Selection: Go to homepage and select a program length using the new card design
  4. Daily Tasks: Complete different types of tasks (timer, journal, checklist)
  5. Progress Tracking: Watch progress bar and circles update
  6. Day Locking: Try to access next day before completing current one
  7. Demo Mode: Use fast-forward button to skip days
  8. Completion: Complete all days to see celebration screen
  9. Persistence: Refresh page to ensure progress is saved

Expected Behavior

  • Only one new day should unlock per calendar day (unless in demo mode)
  • Tasks must meet minimum requirements to enable "Complete Day" button
  • Progress should persist between page refreshes
  • All interactive components should work smoothly
  • Mobile design should be responsive and touch-friendly
